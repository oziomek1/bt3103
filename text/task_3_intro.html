Basic skeleton of a cell class: 

class Cell(Square):
    def __init__(
        self,
        cell_size=20,
        color=(255, 255, 255),  #RGB value for the color white
        x_position=None,
        y_position=None,
        display_window=None):

        self.cell_size = cell_size
        self.x_position = x_position
        self.y_position = y_position
        self.color = color
        self.display_window = display_window

    def draw(self):
        pg.draw.rect(
            self.display_window,
            self.color,
            pg.Rect(self.x_position, self.y_position, self.cell_size, self.cell_size)

Calling methods with objects

Line 1: exampleCell = Cell()
Line 2: exampleCell.draw()

Line by line breakdown

  • To instantiate a class, simply call the class as if it were a function, passing the arguments that the __init__() method requires. The return value will be the newly created object.
  • Looking at line 1, we are creating an instance of the Cell class and assigning it to variable exampleCell.
  • No parameters were passed into Cell() since the __init__() method for Cell class didn't require any.
  • exampleCell is now an instance of Cell class.
  • Looking at line 2, we can now access Cell class methods with exampleCell variable. Hence, we can call draw method with simply calling exampleCell.draw().
