  • An example of the for-loop which incorporates enumerate

    for index, snake_cell in enumerate(self.snake.cells):
    • index here refers to the count for each cell of the body and it starts from 0.
    • snake_cell is the Cell object that corresponds to the index of the list
    • For example if the Game has a Snake object called snake of length 5, using enumerate on snake.cells would assign each cell in snake.cell with an index and the Cell object itself to the variable snake_cell

  • The check_collide() methods would check for 2 conditions

    • First condition: if the snake collides with the boundary of the game

                if snake_cell.x_position < 0 or snake_cell.x_position >= self.window_width or snake_cell.y_position < 0 or snake_cell.y_position >= self.window_height:
      • snake_cell.x_position refers to the x coordinates of the current cell being iterated. If it is less than 0 or greater than window_width it would exceed the screen of the game.

    • Second condition: if the snake collides with a part of itself

                if index > 0 and snake_cell.x_position == head.x_position and snake_cell.y_position == head.y_position:
      • index > 0 here specifies that the current cell musn't be the head of the snake. The next two clauses help to check if the current cell's position is equal to head's position. If it is, the snake's head would have collided with a part of itself.
