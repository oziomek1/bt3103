Instructions

  • Method check_eat checks if the head of the Snake has eaten the Snack in game
  • Remember that the head of the snake belongs to the first element of the cells list 
    self.snake.cells[0]
    • The code above would access the head of the snake
  • Remember that each Cell object has x_position and a y_position attributes. Using this fact, the Snack would be eaten if the x and y position of the snake's head is equals to the x and y position of the Snack.
  • Hence, return True if this if-statement is fulfilled and return False otherwise.
